## Troubleshooting Guide

```
Problem: "CUDA out of memory"
Solution: Use smaller model or CPU
Command: python train.py --model gpt2

Problem: "Invalid JSON generated"
Solution: Add more training data, retrain
Command: python train.py --epochs 5

Problem: "Connection refused"
Solution: Check if ML service is running
Command: python serve.py --model gpt2

Problem: "Slow inference"
Solution: Use quantization or smaller model
Command: python serve.py --model gpt2 --quantize
```
